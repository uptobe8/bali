# Nusa Travel OS v4

Aplicación visual de viajes para Indonesia/Bali, creada como app para usuario final: navegación por pantallas, colores vivos, sonido de selva, mapa interactivo, itinerario día a día, vuelos, alojamientos, actividades, transportes, checklist y decisión final.

## Abrir rápido

1. Descomprime el ZIP.
2. Abre `index.html` con Chrome, Edge o Safari.

## Modo recomendado en Mac

1. Doble clic en `start.command`.
2. Abre `http://localhost:8088`.

## Modo recomendado en Windows

1. Doble clic en `start.bat`.
2. Abre `http://localhost:8088`.

## Notas de funcionamiento

- El mapa usa OpenStreetMap/Leaflet, por lo que necesita internet para cargar teselas reales.
- Las fotos se cargan desde Unsplash. Si no hay conexión, la app mantiene el diseño pero no mostrará todas las imágenes externas.
- El sonido de selva se genera con Web Audio en el navegador. No hay autoplay: se activa solo con botón.
- Los importes son orientativos y están marcados como tales. La disponibilidad real de vuelos, hoteles, ferries y actividades debe confirmarse antes de reservar.
